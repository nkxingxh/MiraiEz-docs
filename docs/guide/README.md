# 介绍

MiraiEz 可以帮助你快速开发自己的 Bot 插件。你可以在左侧浏览更多文档。

::: tip
MiraiEz 是一个 PHP 插件框架，通常不能脱离 mirai-api-http 使用。
:::

欢迎加入我们的交流群: 749709015

## 开始

你可以把 MiraiEz 看作一个 PHP 站点进行部署。
